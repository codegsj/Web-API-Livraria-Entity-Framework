# Web-API-Livraria-Entity-Framework
Desenvolvimento de uma API RESTful para gerenciamento de livros e autores, utilizando C#, .NET e Entity Framework. A API inclui funcionalidades de CRUD e está integrada ao SQL Server. A estrutura segue boas práticas de desenvolvimento, com separação em camadas (Controllers, Services, DTOs e Models). Os endpoints foram testados utilizando o Postman.
